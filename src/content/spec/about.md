## 关于 Gyoza

Gyoza 是一个使用 Astro 和 React 开发的博客主题。Gyoza 借鉴了 [Shiro](https://github.com/innei/Shiro) 和一些网站的设计。

Gyoza 的核心理念是简洁，快速，可爱。

- 在 [Markdown 示例](/posts/markdown) 中展示了 Markdown 的渲染样式
- 可以前往 [Gyoza 使用指南](/posts/guide) 了解 Gyoza 的使用方法
